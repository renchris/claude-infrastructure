# Persistent-State Leak Census — claude-infrastructure @ ef7f7a4

**Headline:** One unwired launchd plist (`com.claude.log-rotation`) is the root cause of 3 unbounded logs totalling ~97MB live + ~50MB gz, growing ~18MB/day.
